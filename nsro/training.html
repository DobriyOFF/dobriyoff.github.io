Учебный центр
АНО «ТехПрогресс»

В АНО «ТехПрогресс» реализованы курсы подготовки соискателей к прохождению независимой оценки квалификации в форме профессионального экзамена по программе:

Главный инженер проекта (специалист по организации архитектурно-строительного проектирования) (7 уровень квалификации)

  • лекционные и презентационные материалы по основным направлениям деятельности специалиста по организации архитектурно-строительного проектирования;
  • видеокурс по основному предмету деятельности специалиста по организации архитектурно-строительного проектирования;
  • материалы для самостоятельного изучения;
  • рассмотрение вопросов подробного формирования портфолио для сдачи практической части профессионального экзамена;
  • оценочные средства, по которым проводится независимая оценка квалификации;
  • симулятор экзамена, который позволяет проверить знания в ситуации приближенной к экзаменационной.

Специалист по организации строительства» по квалификации Главный инженер проекта (специалист по организации строительства) (7-й уровень квалификации)

  • лекционные и презентационные материалы по основным направлениям деятельности специалиста по организации строительства;
  • задачи из практики, практические задания;
  • перечень экзаменационных вопросов и ответов к ним;
  • симулятор экзамена, который позволяет проверить знания в ситуации приближенной к экзаменационной.
